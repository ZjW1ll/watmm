## 📖 Sobre o projeto

**WATMM (We Are The Music Makers)** é uma aplicação web desenvolvida para catalogar e explorar álbuns musicais e seus artistas.

Mais do que uma simples biblioteca de músicas, o projeto busca resgatar a ideia de que **a música é uma obra de arte e um artigo de luxo**, valorizando os álbuns como peças únicas, com história, contexto e identidade.

A proposta é oferecer uma experiência visual inspirada em coleções de discos e galerias, distanciando-se da forma descartável como a música muitas vezes é consumida atualmente.

---

# ✨ Funcionalidades

## 🔍 Pesquisa

- Buscar álbuns pelo nome
- Buscar artistas
- Navegação simples e rápida

---

## 💿 Informações dos álbuns

Cada álbum possui informações detalhadas, como:

- Capa do álbum
- Nome
- Artista
- Ano de lançamento
- Quantidade de faixas
- Duração total
- Lista completa das músicas

---

## 🎤 Informações dos artistas

Cada artista possui uma página dedicada contendo:

- Nome
- Foto
- Biografia
- Data de nascimento
- Data de falecimento (quando aplicável)
- Discografia presente no sistema

---

## ⚙️ Painel administrativo

O projeto conta com uma interface para gerenciamento do banco de dados.

É possível realizar o cadastro de:

- Artistas
- Álbuns
- Faixas (Tracks)

Sem necessidade de inserir informações manualmente no banco de dados.

---

# 🎨 Conceito visual

O WATMM foi pensado para transmitir a sensação de estar visitando uma coleção de discos raros.

A identidade visual utiliza:

- tons escuros
- vermelho
- bordô
- detalhes dourados
- inspiração retrô

A intenção é fazer com que cada álbum seja tratado como uma peça de coleção, e não apenas como um arquivo digital.

# 💡 Motivação

Hoje a música costuma ser consumida de forma extremamente rápida.

O álbum, que antes era uma experiência completa, muitas vezes foi reduzido a playlists e faixas isoladas.

O WATMM nasce com o objetivo de valorizar novamente o conceito de álbum como uma obra artística, incentivando o usuário a conhecer sua história, contexto e criação.

Mais do que organizar músicas, o projeto busca preservar um pedaço da cultura musical.
